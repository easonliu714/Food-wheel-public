🍴 今天吃什麼？公開版

這是一個開源的美食輪盤工具。為了免費且長久運作，本工具採用 Bring Your Own Key 模式。請輸入您自己的 Google Maps API Key 即可開始使用。

🚀 初次使用？請依照步驟申請金鑰：

1️⃣

建立專案

前往 Google Cloud Console，登入 Google 帳號並建立一個新專案。

💳

綁定結算帳戶 (重要)

前往「計費 (Billing)」綁定信用卡。Google 每月提供 $200 美金免費額度 (約可搜尋 10,000 次)，個人使用通常完全免費，但未綁卡無法使用 API。

🗺️

啟用必要 API

前往「API 和服務」 > 「啟用 API」，搜尋並啟用以下 4 個服務：

  • ✅ Maps JavaScript API
  • ✅ Places API (搜尋店家用)
  • ✅ Geocoding API (地址解析用)
  • ✅ Distance Matrix API (計算路程用)
🔑

取得 API Key

前往「憑證 (Credentials)」建立 API Key。建議設定「網址限制」為您的 GitHub Pages 網址以策安全。

🔒 您的 Key 僅儲存於您瀏覽器的暫存區 (LocalStorage)，不會傳送給開發者，請安心使用。
